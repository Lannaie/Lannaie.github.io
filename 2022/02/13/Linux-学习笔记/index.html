Linux - 学习笔记

发表于 2022-02-13 更新于 2022-03-19 分类于 刷题 ， 学习笔记

ps -ef 和 ps aux

都用来显示全部进程，-ef 显示标准格式，aux 显示BSD格式。

grep

查找输入的内容中符合指定内容的字符串。

grep [指定内容] [输入内容，一般此处放文件]

ps -ef | grep [指定内容]

显示所有进程中与指定内容相同的进程。

grep -v

grep -v grep：取出不包含grep的进程行。

grep -v "name"：取出不包含字符串 name 的行。

wc

计算字数。

wc -l

显示行数

wc -L

显示当前行的长度。

nohup

不挂断的执行命令。

nohup &

在后台执行。会返回一个进程号用于查该进程。

netstat

查看网络相关的信息。

netstat -ap | grep [端口号]

查看指定端口有哪些进程。

kill

删除执行中的程序或工作。

kill -9

强制杀死进程。

统计文件行数

两种方式：

1. wc -l test.txt | awk ‘{print $1}’

wc -l test.txt 显示行数和 ‘test.txt’ 两列。

awk '{print $1}' 显示管道前输出的第一列。

2. awk ‘{print NR}’ test.txt | tail -n1

awk '{print NR}' test.txt 显示 test.txt 文件的行号，如：

tail -n 1 最后一行。

3. awk ‘END{print NR}’ test.txt

END 必须大写。

4. grep -c “” test.txt

grep -c 输出文件 test.txt 中匹配字符串 "" 的行。

5. grep -n “” test.txt | awk -F “:” ‘{print $1}’

grep -n "" test.txt 输出文件 test.txt 中匹配字符串 "" 的行号及行，格式为：行号:行内容 ；

awk -F ":" '{print $1}' 将输入依照 : 分隔符切分，并输出第一列。

awk内建变量

RS 指定行分隔符，使用 awk 'BEGIN{RS="|"}{print $0}' ，将一行按照分隔符 | 分割为多列并显示。

sort 语法：

  6. -k ：按第几列排序。

print

是ksh的内置命令，不能使用 %s、%d、%c 等占位符，会自动换行。

printf

是bash的内置命令，支持使用占位符，不会自动换行，所以要换行的时候需要使用 \n 。

常见的：**%s %c %d %f** 都是格式替代符，％s 输出一个字符串，％d 整型输出，％c 输出一个字符，％f 输出实数，以小数形式输出。
