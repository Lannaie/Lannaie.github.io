解题思路

哈希表

使用 HashSet 存已经遍历过的节点，第一个二次遍历的节点就是入口结点。

快慢指针

这道题的重点是获取入口节点，但是快慢指针重合的第一个节点不一定是这个入口结点。下面进行简单的推导：

[上图引用自官方题解]

上图假设快慢指针在紫色圆点出重合，此时慢指针走过 a + b，快指针走过 a + n(b + c) + b。
$$
\begin{cases}
fast = a + n(b + c) + b\
slow = a + b\
fast = 2 * slow
\end{cases}
$$
上述方程组可以推导出：
$$
\begin{align*}
&\qquad a + n(b + c) + b = 2(a + b)\
&=> a + (n + 1)b + nc = 2a + 2b\
&=> a = c + (n - 1)(b + c)
\end{align*}
$$
也就是，将指针从 head 开始，head = head.next; slow = slow.next ，这二者第一次重合的节点为入口结点。
