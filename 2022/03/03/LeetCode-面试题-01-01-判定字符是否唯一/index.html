题目

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。

示例 1：

输入: s = “leetcode”
输出: false

示例 2：

输入: s = “abc”
输出: true

限制：

0 <= len(s) <= 100
如果你不使用额外的数据结构，会很加分。

位运算

第二个思路就是将字符转换为数字，然后计算数字出现的个数，又考虑到不使用其他的数据结构，联想到位运算。

  1. 使用一个 mark = 0 数字的二进制作为数组；
  2. 计算 bit = astr[i] - 'a' ，为了让 mark 在相应的二进制位能够置1，取 1 << bit 得到相应位置的1，再 mark |= (1 << bit) 在相应位置置1，只要 mark 在该位置为0，就说明没有重复。
