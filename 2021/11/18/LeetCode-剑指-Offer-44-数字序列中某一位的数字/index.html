LeetCode - 剑指 Offer 44. 数字序列中某一位的数字

发表于 2021-11-18 更新于 2022-01-28 分类于 刷题 ， 算法

题目

https://leetcode-cn.com/problems/shu-zi-xu-lie-zhong-mou-yi-wei-de-shu-zi-lcof/

算法思路

插值查找是二分查找的变种。

二分查找是：有三个变量，low = 0， high = num.length, mid = (low + high)/2，这三个变量是固定的，插值查找的这三个变量是可变。

将 low 和 high 当成是上限和下限， mid 是一个灵活的具有规律的指定元素，可以分析该题：

  1. 字符串可以分为多个区间：0-9、10-99、100-999...... ；
  2. 因为 0-9 返回本身，所以直接返回可以省去一次运算；
  3. 接下来降维计算。像这种求区间中的具体值的，可以考虑这种方法；
  4. 因为是一维一维降的，所以不符合if条件的肯定会降下第一维：n -= 9 ；
  5. 之后从第二维开始：start = 10, end = 99, digit = 2 ；
  6. 开始一维一维的减，直到n 在某一维区间内，退出循环；
  7. 计算该区间的该 n 对应的实际数字：(start + (n - 1) / digit) ，该区间从 start 开始，n - 1 是因为每次以 0 为首个下标，所以需要 - 1 ，为了方便取数，+"" 变成字符串；
  8. 之后 (n - 1) % digit 取哪一位上的数。
